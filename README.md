## 📂 Featured Projects
#### [Haver & Boecker Industrial Order Tracking System](https://github.com/Crashjoy/Sponsored-Project)🔒 (Private Repository)
- **The Problem:** Industrial workflows had high data-entry error rates and slow processing times.
- **The Solution:** Engineered a scalable C#/.NET MVC system integrated with existing ERP systems via RESTful APIs.
- **Key Tech:** C#, ASP.NET MVC, SQL Server, Entity Framework.
- **Impact:** Reduced order-processing time by 25% and decreased data-entry errors by 20%.

#### [KostMoving - AI-Powered Web Solution](https://github.com/Crashjoy/Moving_Project)🔒 (Private Repository)
- **The Problem:** A local moving business needed to automate inventory assessment and quote generation.
- **The Solution:** Developed a custom web platform integrating a Python-based YOLO model for automated image recognition of inventory.
- **Key Tech:** Python, ASP.NET Core, SQLite, Google Maps API.
- **Impact:** Automated customer engagement and provided accurate, distance-based pricing models.

#### [Niagara College Gym Management System](https://github.com/Crashjoy/Gym-Management-Project)
- **The Problem:** Administrative overhead for tracking member schedules, instructor access, and reporting.
- **The Solution:** Built a full-stack management application featuring role-based authentication (ASP.NET Identity) and automated administrative reporting.
- **Key Tech:** ASP.NET Core MVC, SQLite, Bootstrap, Entity Framework.
- **Impact:** Streamlined facility operations by implementing secure access control and automated Excel report generation for staff.
